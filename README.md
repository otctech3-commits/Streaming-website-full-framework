📚 How To Add More Content

*🎯 Quick Add - 30 Seconds*

*Step 1:* Open `app.js`  
*Step 2:* Find `animeData` array at the top  
*Step 3:* Copy this template and paste it *before* the closing `];`
{
  id: 9, // Change to next number: 10, 11, 12...
  title: "Demon Slayer Season 4",
  category: "action", // action, adventure, comedy, drama, fantasy, romance, sci-fi
  type: "tv", // movie, tv, ova
  year: 2024,
  rating: 9.1,
  duration: "12 ep", // "1h 45m" for movies, "24 ep" for series
  image: "https://your-cover-image.jpg", // 400x600px works best
  description: "Tanjiro returns to face new demons in the Hashira Training Arc. Full description here...",
  videoUrl: "https://www.youtube.com/embed/YOUR_VIDEO_ID", // Get from YouTube share → embed
  episodes: 12, // Number of episodes. Use 1 for movies
  trending: false // Set to true to show in hero banner. Only 1 can be true.
},
*Step 4:* Save → Commit → Auto-deploys in 30 sec

*🎬 Where To Get Video URLs*
Platform	How To Get Embed Link
**YouTube**	Share → Embed → Copy `src` URL only
**Vimeo**	Share → Embed → Copy `src` URL only
**Google Drive**	Get shareable link → Replace `/view` with `/preview`
**Self-Hosted**	Upload MP4 → Use direct `.mp4` link
*Example:* `https://www.youtube.com/embed/dQw4w9WgXcQ`

*🖼️ Where To Get Cover Images*

1. *MyAnimeList* - Right click poster → Copy image address
2. *AniList* - Click anime → Right click cover → Copy image
3. *Imgur* - Upload → Copy direct link
4. *Unsplash* - Search "anime" → Free to use

*Best size:* 400x600px or 2:3 ratio

*📊 Understanding All Fields*
{
  id: 9,              // Unique number. Must be different for each anime
  title: "...",       // Shows on card + player
  category: "action", // Used for genre filter: action/adventure/comedy/drama/fantasy/romance/sci-fi
  type: "tv",         // Used for type filter: movie/tv/ova
  year: 2024,         // Used for year filter
  rating: 9.1,        // Shows star rating. 0-10
  duration: "12 ep",  // Shows on card. "1h 45m" for movies
  image: "...",       // Cover poster URL
  description: "...", // Shows in player modal
  videoUrl: "...",    // Embed URL for video
  episodes: 12,       // If >1, shows episode selector
  trending: false     // If true, shows in hero banner at top
}

🎨 Customization Guide

*1. Change Logo & Name*
`index.html` line 28:
<span class="logo-text">𝙈𝙆<span>ANIME</span></span>
<!-- Change to: -->
<span class="logo-text">YOUR<span>NAME</span></span>
*2. Change Colors*
`style.css` line 3:
:root{
  --primary:#e94560;  /* Main brand color */
  --accent:#16db65;   /* Secondary color */
}
Replace hex codes with your colors.

*3. Change Hero Background*
`app.js` function `updateHero()` - It auto-uses the `trending: true` anime image.

*4. Add Your Social Links*
`index.html` - Find footer social links and replace `#` with your URLs.

🐛 Troubleshooting
Problem	Solution
**Video not playing**	Check if `videoUrl` is embed link, not watch link
**Image broken**	Use direct image URL ending in `.jpg` or `.png`
**Not showing**	Check `id` is unique and no syntax errors in `app.js`
**Deploy failed**	Make sure file names are exactly `index.html` `style.css` `app.js`
